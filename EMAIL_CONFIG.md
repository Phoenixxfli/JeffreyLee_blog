## 什么时候需要配置邮件？

如果以后要添加以下功能，才需要配置邮件服务：

### 1. 密码重置功能
- 用户忘记密码时，发送重置链接到邮箱

### 2. 邮箱验证功能
- 注册后需要验证邮箱才能使用

### 3. 通知功能
- 新评论通知
- 新留言通知等

## 如何配置邮件服务（未来需要时）

### 方案 1：使用 Resend（推荐）⭐

**优点**：
- 免费额度：每月 3000 封邮件
- 配置简单
- 送达率高

**步骤**：
1. 注册 Resend：https://resend.com
2. 创建 API Key
3. 验证域名（可选，但推荐）
4. 在 Vercel 环境变量中添加：

```
RESEND_API_KEY=re_xxxxxxxxxxxxx
EMAIL_FROM=noreply@yourdomain.com
```

### 方案 2：使用 SMTP（自有邮箱）

**适用场景**：
- 有自己的企业邮箱
- 使用 Gmail、QQ 邮箱等

**步骤**：
1. 获取 SMTP 配置信息
2. 在 Vercel 环境变量中添加：

```
EMAIL_SERVER=smtp://username:password@smtp.example.com:587
EMAIL_FROM=noreply@example.com
```

**示例（Gmail）**：
```
EMAIL_SERVER=smtp://yourname@gmail.com:app_password@smtp.gmail.com:587
EMAIL_FROM=yourname@gmail.com
```

**注意**：Gmail 需要使用"应用专用密码"，不是普通密码。
