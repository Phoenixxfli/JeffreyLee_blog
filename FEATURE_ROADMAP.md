# 博客功能完善路线图

## 📋 已完成功能

- ✅ 基础博客功能（文章列表、详情、标签、归档）
- ✅ WYSIWYG 编辑器（支持图片/视频/音频上传）
- ✅ 文章创建和编辑（Markdown 编辑器）
- ✅ 文章删除功能（在编辑页面）
- ✅ 留言板（Guestbook）
- ✅ 用户认证（用户名/密码注册登录）
- ✅ 管理员权限系统
- ✅ Giscus 评论系统（基于 GitHub Discussions）

## 🚀 待实现功能（按优先级）

### 🔴 高优先级（核心功能）

#### 1. 自定义评论系统
- **数据库模型**：创建 `Comment` 模型
  - 字段：id, postId, userId, parentId（回复）, content, status, createdAt
- **评论 UI**：在文章详情页添加评论表单和评论列表
- **评论管理**：Admin 后台评论管理页面
- **功能**：支持回复、审核、删除

#### 2. 编辑页面改用 WYSIWYG
- 将 `/admin/posts/[id]` 页面的编辑器改为 WYSIWYG
- 与新建页面保持一致的用户体验
- 支持加载现有 HTML 内容

#### 3. 文章列表优化
- 在 `/admin/posts` 页面添加快速删除按钮
- 添加批量操作（批量删除、批量发布/下架）
- 添加筛选功能（按状态、标签、日期）

### 🟡 中优先级（增强功能）

#### 4. 文章搜索
- 首页搜索框优化（当前已有基础搜索）
- Admin 后台文章搜索
- 支持按标题、内容、标签搜索

#### 5. 文章统计
- 阅读量统计（每篇文章的访问次数）
- 评论数统计（显示在文章卡片上）
- 在 Admin 后台显示统计数据

#### 6. 标签管理
- 创建标签管理页面 `/admin/tags`
- 支持标签的增删改查
- 显示每个标签下的文章数量
- 标签颜色/图标设置

#### 7. 媒体库
- 创建媒体库页面 `/admin/media`
- 显示所有上传的图片/视频/音频
- 支持删除、查看使用情况
- 支持直接插入到编辑器

### 🟢 低优先级（优化功能）

#### 8. 草稿预览
- 允许管理员预览草稿文章
- 添加预览链接功能

#### 9. SEO 优化
- 为每篇文章添加动态 meta tags
- Open Graph 标签
- Twitter Card 标签
- 结构化数据（JSON-LD）

#### 10. 文章导出
- 支持导出为 Markdown
- 支持导出为 PDF（可选）

#### 11. 用户管理
- Admin 后台用户管理页面
- 查看用户列表、删除用户
- 修改用户角色

#### 12. 通知系统
- 评论回复时发送邮件通知（可选）
- 新评论通知管理员

#### 13. 数据备份
- 数据库备份功能
- 数据恢复功能
